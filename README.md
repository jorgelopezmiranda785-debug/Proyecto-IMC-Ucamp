# Proyecto-IMC-Ucamp
Proyecto final del curso de introducción a Phyton por Ucamp

Este proyecto es un programa en Python que solicita datos personales del usuario, valida que la información sea correcta y calcula el Índice de Masa Corporal (IMC). También muestra la clasificación correspondiente según el resultado obtenido: bajo peso, peso normal, sobrepeso u obesidad.

## Funcionalidades del programa

- Captura de datos del usuario:
  - Nombre
  - Apellido paterno
  - Apellido materno
  - Edad
  - Peso
  - Estatura

- Validación de datos:
  - No permite campos vacíos
  - Verifica que edad, peso y estatura sean valores numéricos

- Cálculo del IMC utilizando la fórmula:
  
  IMC = peso / estatura²

- Clasificación del IMC:
  - Bajo peso
  - Peso normal
  - Sobrepeso
  - Obesidad

- Impresión de todos los datos ingresados junto con el IMC y su clasificación
